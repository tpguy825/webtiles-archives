JS

(click and scroll)

Your JavaScript runs in a sandboxed interpreter with limited APIs. Only ES5 is available.

The following features are not supported:

  • let / const (use var)
  • Arrow functions =>
  • Template literals \`...\`
  • Classes, destructuring, spread operator
  • async / await, Promises
  • Modules (import / export)

Available APIs:

  • DOM — no innerHTML setter, element count limits, sandboxed to your tile
  • Canvas 2D — full drawing support with size limits
  • XMLHttpRequest — external URLs only (no same-origin or relative requests)
  • Events — addEventListener, click, keyboard, mouse, touch and other events.
  • Timers — setTimeout, setInterval
  • localStorage — limited storage size
  • location, DOMParser, atob/btoa, console, alert/prompt/confirm
